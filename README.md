# mmyScripts
Plugin for SuperMasao PluginManager.js

JSまさおに新たな仕掛けとアクションを追加します。

## 正男の特技

|番号|内容|
|---|---|
|101|Xキーでスピン攻撃|
|102|HP表示 3|
|103|HP表示 5|
|104|重力を上にする|
